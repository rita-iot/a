# Docsify部署教程

安装教程

**软件环境**

- nodejs

## 安装过程

推荐全局安装 docsify-cli 工具，可以方便地创建及在本地预览生成的文档。

```javascript
npm i docsify-cli -g
```

文件初始化

```javascript
docsify init ./Docsify-Guide
```

运行项目

```javascript
docsify serve Docsify-Guide
```
